.. _reference

Identifiers
===========
**[identifiers]**

This section describes the identifier conventions used by |short_name|.

Case
----

Identifier conventions in the library follow ISO C++ standard library style.
Identifiers use underscore_style, while concepts use PascalCase.

Reserved Identifier Prefixes
----------------------------

The library reserves the ``__TBB`` prefix for internal identifiers and macros that must
not be referenced by user code.
